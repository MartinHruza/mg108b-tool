# Documentation

Reverse-engineering documentation for the MonsGeek MG108B keyboard, produced by
analyzing the official MonsGeek Driver v1.0.3.6 Electron app and the firmware
binary.

- [driver.md](driver.md) — USB HID protocol documentation (commands, payloads, checksum format)
- [firmware.md](firmware.md) — Firmware binary analysis (YiChip YC31xx, memory layout, key structures)
